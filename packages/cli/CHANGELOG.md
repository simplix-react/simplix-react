## [0.0.1] - 2025-06-01

### Added

- `simplix init` command for scaffolding new simplix-react projects
- `simplix add-domain` command for generating domain entity boilerplate
- `simplix add-module` command for generating feature module boilerplate
- `simplix validate` command for verifying project structure and conventions
- `simplix i18n-codegen` command for generating type-safe translation keys
- `simplix migration` command for managing PGlite mock database migrations
- `simplix openapi` command for importing contracts from OpenAPI specifications
- Built on Commander.js with interactive prompts (prompts), Handlebars templates, and colored output (picocolors)
