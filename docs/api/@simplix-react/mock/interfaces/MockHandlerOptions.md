### responseWrapper?

> `optional` **responseWrapper**: [`MockResponseWrapper`](MockResponseWrapper.md)

Defined in: [derive-mock-handlers.ts:85](https://github.com/simplix-react/simplix-react/blob/main/derive-mock-handlers.ts#L85)

Wraps every mock response in a server-specific envelope format.

By default, success responses return `{ data: <payload> }` and error
responses return `{ code, message }`. Use this to match a custom backend
envelope (e.g. SimpliX Boot's `{ type, message, body, timestamp }`).
